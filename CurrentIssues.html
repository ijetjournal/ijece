5G: A New Air Interface With Radio Access Proficiencies and Technologies


Authors: Pritam Singh Patel, Ravendra Ratan Singh, Satya Patra

  • [ Abstract ] [ cite ] [ PDF ]

    • Abstract :-

    5G radio access technology will be a key component of the Networked Society and is expected to extend beyond previous generations of mobile communication. The intention is to address high traffic growth and increasing demand for high-bandwidth connectivity by providing higher data rates and ultra-high reliability while lowering the latency and energy consumption. This can be realized by combining new radio access technologies with LTE relying on key technology components such as ultra-lean design, multi antenna transmission and extension to higher frequency bands, access/backhaul integration, device-to-device communication, flexible duplex and flexible spectrum usage.

    Keyword: LTE, HSPA, Radio Access Technology, GSM, IoT, Machine-type communication, Frequency Division Duplex.

Design of Visible Light Communication System for Various Applications Using LED


Authors: S.M. Hindoliya

    [ Abstract ] [ cite ] [ PDF ]

    Abstract :-

    The communication technology where the visible light acts as a signal transmitter, the transmission medium is the air and the signal receiving component is a photodiode which is appropriate is referred as VLC. VLC which also known as Visible Light Communication is one of the keenly discussed topic amongst the scientific community. The reason for this is its enormous role in creation of smart communication links. It is possible to get internet connectivity with light lamps where VLC plays a crucial role. This thesis considering the illumination, discusses about LED characteristics and data format so that the VLC link configuration can be commercialized.
    Key word :- Visible light communication, light emitting diode, Photodiodes, IR communication, VLC Applications.
